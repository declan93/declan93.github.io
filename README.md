# Declan Bennett - Personal Website

Personal academic website for Declan Bennett, Computational Biologist at St. Jude Children's Research Hospital.

## About

This site showcases my research in computational biology and pediatric oncology, including:

- **Pediatric Dependency Map (PedDep)** - Computational analysis of ~300 pediatric cancer cell lines
- **ALTitude** - Machine learning framework for predicting Alternative Lengthening of Telomeres (ALT) status
- **Cancer Genomics** - Somatic mutation analysis, GWAS methods, and large-scale genomic studies

## Built With

- [Jekyll](https://jekyllrb.com/) - Static site generator
- Hosted on [GitHub Pages](https://pages.github.com/)

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve
```

Visit `http://localhost:4000` to preview the site.
